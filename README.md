# 🎮 Family Feud Broadcast System (Python + Pygame + Tkinter)

Sistem game show **Family Feud / Family Fortunes** versi broadcast yang terdiri dari:

- 🎛️ Editor PRO (buat soal + Fast Money)
- 🎮 Game Display (Main Game + Fast Money)
- 🎛️ Host Control Panel (control live)
- 🖥️ Dual Screen Support (Host & Display terpisah)
- 🧩 Asset-based UI (PNG → mirip TV)

---

# 🚀 FITUR UTAMA

## 🎮 MAIN GAME
- Maksimal 8 jawaban
- Sistem score + target
- Strike (X) system
- Reveal manual (host control)
- Multi-round (Single / Double / Triple / Random)

## 💰 FAST MONEY
- 5 pertanyaan
- Input jawaban real-time
- Timer
- Total score (target ≥ 200)

## 🎛️ HOST CONTROL PANEL
- Reveal jawaban (1–8)
- Tambah strike
- Next round
- Trigger Fast Money
- Reset game

## 🧠 EDITOR PRO
- Multi round editor
- Fast Money editor
- Live preview board
- Auto save JSON (realtime)

## 🔄 LIVE SYSTEM
- Tanpa restart game
- Auto reload JSON
- Sinkronisasi via file:
  - `soal_live.json`
  - `control.json`
